# Spacer Mission Control - Stakeholder Document

## Project Overview
**Spacer Mission Control** is an interactive web application for space exploration enthusiasts. It provides real-time telemetry of satellites, ISS, asteroids, and NASA updates, alongside educational insights on the solar system and near-Earth objects.

## Objectives
- Educate users about space science, satellites, and NASA missions.
- Provide real-time data for ISS, NEOs, and daily NASA images.
- Offer an AI-powered assistant for space-related queries.

## Key Features
- **Dashboard**: Displays active satellites, ISS altitude, and NEO statistics.
- **Solar System Module**: Planet data, exploration highlights, and telemetry.
- **NASA Image of the Day**: Daily space imagery and explanations.
- **ISS Tracker**: Live ISS altitude and map.
- **Asteroid Monitor**: Catalog of NEOs, planetary defense highlights.
- **AI Chatbot**: Answer space-related queries using N8N integration.

## Stakeholders
- **Primary Users**: Students, educators, space enthusiasts.
- **Developers**: Athri / Team (current developer).
- **Content Providers**: NASA, ISS telemetry sources.
- **Investors / Sponsors**: Educational institutions, tech partners.
